How to run the program

0. linux env

1. pip install pybind11

2. bash build.sh

3. Use myalgorithm.py

-----------------------------
** You must have g++, bits/stdc++.h and openmp
** If you want to compile and run on windows, use MinGW to download g++ and change build.sh accordingly.
** If you want to compile and run on macOS, use homebrew to download g++ and change build.sh accordingly.
